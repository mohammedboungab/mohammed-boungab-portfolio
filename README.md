# Portfolio — Mohammed Boungab

Portfolio professionnel construit avec Next.js (App Router), TypeScript, Tailwind CSS, des composants shadcn/ui et Lucide Icons.

## Démarrage

```bash
pnpm install
pnpm dev
```

Puis ouvrir `http://localhost:3000`.

## Vérifications

```bash
pnpm lint
pnpm typecheck
pnpm build
```

## URL publique

Renseigner `NEXT_PUBLIC_SITE_URL` dans l’environnement de déploiement avec l’URL publique finale, sans barre oblique terminale. Tant que cette variable est absente en production, les URL canoniques et sociales ne sont pas publiées, le sitemap reste vide et l’indexation est désactivée.
